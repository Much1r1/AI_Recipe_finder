# AI-powered personalized food decision system
An AI system that helps people decide what to cook based on their ingredients, time, budget, and nutrition goals

# Success metric
Percentage of searches where a user selects a recipe without refining the query
Time-to-usable recipe <= 15 seconds
Time-to-first-cook (TTFC): How long it takes a user from search -> starting to cook

# Target
Busy professionals, later I'll include more target groups
